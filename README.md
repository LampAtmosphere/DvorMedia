# Информационная система ДВОР

"Информационная система ДВОР" — это приложение, предназначенное для создания, отслеживания мероприятий, которые проводятся в «Двор`е», а также для ведения посещаемости.

## Цель

Разработать удобный инструмент для создания, отслеживания мероприятий и ведения статистики посещаемости.

## Возможности

- Создание мероприятий
- Просмотр мероприятий
- Просмотр и создание списка пришедших сегодня
- Просмотр и создание общего списка всех людей

## Технологии

- Язык программирования: Kotlin
- База данных: Firebase

## Логирование

Для логирования в проекте используется библиотека SLF4J, которая предоставляет следующие преимущества:

- Единый набор API для логирования.
- Возможность настройки различных режимов (отладка и релиз).
- Эффективное выполнение при отключенном логировании.
- Простая интеграция с другими библиотеками логирования.

SLF4J используется для отслеживания событий, запуска приложения и базовых операций, связанных с базой данных Firebase.
